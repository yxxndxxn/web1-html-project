태그

“img, input, br, hr, meta” 는 닫지 않는 태그

  • !DOCTYPE html : 이 웹 페이지가 html로서 만들어졌다는 것을 표현하기 위한 코드
  • html /html : body 태그와 head 태그를 감싸는 하나의 태그
  • head /head : 본문을 설명하는 태그
  • title /title : 웹 페이지 제목 지정
  • meta charset="utf-8" : 웹브라우저야! 이 웹페이지는 utf-8로 만들어졌기 때문에 utf-8로 열어야 해~~
  • body /body : 본문
  • h1 ~ h6 : 숫자가 클수록 글씨가 작아짐
  • strong /strong : 볼드체
  • u /u : 밑줄
  • p /p : 단락 표현 (단락 간격이 고정되어 있음-> css로 해결 가능)
  • br : 줄 바꿈 (닫지 않는 태그)
  • li /li : 목차
  • ul /ul : 다른 목록과 구분할 수 있도록 하는 경계 (ul = unordered list의 약자)
  • ol /ol : 번호 매겨줌 (ol = ordered list의 약자)
  • a /a : 링크 걸 때
  • href는 링크 주소 첨부 속성
  • target=”_blank”는 링크를 클릭했을 때 새 창에서 페이지가 열리게 하는 속성
  • title은 이 링크가 어떤 내용을 담고 있는지 툴팁으로 보여주는 기능
